# teacher-s-journal
Школьный журнал учителя, написанный на Java 11. 

Приложение имеет функционал классического школьного журнала учителя, позволяет фиксировать успеваемость учеников.

## Список используемых технологий ##
*Java 11;
*Spring Boot 2.0;
*Spring MVC;
*Spring DataJPA;
*Spring Security;
*Thymeleaf;
*ORM Hibernate;
